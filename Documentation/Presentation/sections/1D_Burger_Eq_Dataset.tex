\section{1D Burgers Equation Dataset}

\begin{secframe}

\begin{block}{Dataset Generation}
\begin{itemize}
    \item We generated a dataset with different viscosity values $\nu$.
    \item Our training dataset consists of \textbf{724 trajectories} across \textbf{13 different viscosity values} (from $0.001$ to $0.5$).
    \item Our testing dataset consists of \textbf{123 trajectories} with \textbf{4 new viscosity values} to test generalization.
    \item Each trajectory corresponds to the evolution of the Burger's equation across $128$ spatial positions and over $256$ time steps.
    \item The higher the value of $\nu$, the smoother the solution is over time.
    \item The dataset was generated using a custom simulator implemented in Python.
    \item The simulator uses different types of initial conditions to create a diverse set of trajectories.
\end{itemize}
\end{block}
\end{secframe}
